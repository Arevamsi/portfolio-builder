Flask Resume Generator

This is a simple Flask-based web application that generates a basic resume using user-provided details. The user enters personal, educational, and professional information through a form, and the app dynamically creates a formatted resume.

Description

The application collects inputs such as name, email, skills, education, experience, and target job role. Using Flask’s routing and template rendering, it generates a resume text and displays it on a result page.

Features

Web-based resume generator

Uses Flask framework

Accepts user input through HTML forms

Dynamically generates resume content

Beginner-friendly Flask project

Technologies Used

Python

Flask

HTML (Jinja2 templates)

Project Files

app.py – Main Flask application

templates/index.html – Form for entering resume details

templates/result.html – Displays the generated resume
